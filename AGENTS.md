# AGENTS.md - Reglas para este proyecto

## Regla obligatoria: Verificar antes de continuar

Cada vez que el usuario indique que ha completado un paso o tarea:

1. **Siempre verifica** el contenido del archivo(s) afectado(s) usando `read` antes de dar el siguiente paso
2. **Confirma** que el archivo está correcto y completo
3. **Solo entonces** procede al siguiente paso

Nunca asumas que un archivo se creó correctamente sin leerlo primero.
